# getter

## 1. **Effects**

1. 创建仓库 清除历史记录 恢复仓库
2. 添加文件到暂存区
3. 将暂存区的修改提交到本地Git仓库
4. 将本地代码库的更改推送到远程仓库

2. [git-add][git-add]
3. [git-commit][git-commit]
